from fastapi import APIRouter, Request, HTTPException, Query

**Endpoint: GET /neighbourhoods (API-01):**
- Return pre-serialized GeoJSON bytes from `request.app.state.geojson_bytes`
- Set Content-Type to `application/geo+json`
- Implementation: `return Response(content=request.app.state.geojson_bytes, media_type="application/geo+json")`

**Endpoint: GET /neighbourhoods/{nid} (API-02):**
- Accept `nid` as string path parameter
- Zero-pad to 3 digits: `nid = str(nid).zfill(3)`
- If `nid not in request.app.state.valid_nids`: raise `HTTPException(status_code=404, detail=f"Neighbourhood {nid} not found")`
- Build NeighbourhoodDetail response from app.state:
  - `vibe_scores` from `app.state.vibe_scores[nid]`
  - `dominant_vibe` = `max(vibe_scores, key=vibe_scores.get)`
  - `dominant_vibe_score` = `vibe_scores[dominant_vibe]`
  - `topics` from `app.state.topics.get(nid, [])`
  - `quotes` from `app.state.quotes.get(nid, {})`
  - `review_count` from `app.state.review_counts.get(nid, 0)`
  - `neighbourhood_name` from `app.state.nid_to_name.get(nid)` -- already built at startup by loader.py (Plan 01), no extra lookup needed
- Return as NeighbourhoodDetail model

**Endpoint: GET /temporal (API-03):**
- Return `request.app.state.temporal` directly as JSON (FastAPI auto-serializes dict)

**Endpoint: GET /similar (API-04):**
- Query params: `id: str`, `k: int = Query(default=5, ge=1)`
- Zero-pad id: `nid = str(id).zfill(3)`
- If `nid not in request.app.state.valid_nids`: raise `HTTPException(status_code=404, detail=f"Neighbourhood {nid} not found")`
- Clamp k to max: `k = min(k, len(request.app.state.valid_nids) - 1)`
- Build query vector: `archetype_order = ["artsy", "foodie", "nightlife", "family", "upscale", "cultural"]`; `query = np.array([[app.state.vibe_scores[nid][a] for a in archetype_order]], dtype=np.float32)`
- Normalize: `faiss.normalize_L2(query)`
- Search: `D, I = app.state.faiss_index.search(query, k + 1)`
- Build results, excluding self: for each (dist, idx) in zip(D[0], I[0]), map idx to nid via `app.state.faiss_id_map[str(int(idx))]`, skip if matches query nid
- Return first k results as `list[SimilarNeighbourhood]`
