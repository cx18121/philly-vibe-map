## NTA Attribute Fields (Socrata GeoJSON export)
- NTACode — unique identifier string e.g., "MN0101" (use as neighbourhood_id)
- NTAName — official merged name e.g., "Financial District-Battery Park City"
- BoroCode — "1" Manhattan, "2" Bronx, "3" Brooklyn, "4" Queens, "5" Staten Island (all strings)
- BoroName — "Manhattan", "Brooklyn" etc.
- geometry — MultiPolygon, WGS84 EPSG:4326

https://data.cityofnewyork.us/api/geospatial/9nt8-h7nd?method=export&type=GeoJSON
```

"""
01_download_boundaries.py
Download 2020 NTA GeoJSON from NYC Open Data, filter to Manhattan + Brooklyn,
write to data/boundaries/nta_2020_manhattan_brooklyn.geojson.

Usage:
    python scripts/01_download_boundaries.py
"""
from __future__ import annotations

import datetime
import io
import json
import sys
from pathlib import Path

import geopandas as gpd
import requests

GEOJSON_URL = (
    "https://data.cityofnewyork.us/api/geospatial/9nt8-h7nd"
    "?method=export&type=GeoJSON"
)
OUTPUT_DIR = Path("data/boundaries")
OUTPUT_PATH = OUTPUT_DIR / "nta_2020_manhattan_brooklyn.geojson"

_ANSI = {"WARN": "\033[93m", "FAIL": "\033[91m", "INFO": ""}
_RESET = "\033[0m"

def _log(level: str, msg: str) -> None:
    ts = datetime.datetime.utcnow().strftime("%Y-%m-%d %H:%M:%S")
    color = _ANSI.get(level, "") if sys.stdout.isatty() else ""
    reset = _RESET if color else ""
    print(f"{color}[{level}] {ts}  {msg}{reset}", flush=True)


def download_nta_boundaries(output_path: str) -> gpd.GeoDataFrame:
    """Download full 5-borough NTA GeoJSON, filter to Manhattan (1) and Brooklyn (3).

    Args:
        output_path: Where to write the filtered GeoJSON.

    Returns:
        Filtered GeoDataFrame (EPSG:4326).

    Raises:
        requests.HTTPError: if download fails.
        ValueError: if BoroCode filter returns 0 rows.
    """
    _log("INFO", "Downloading NTA GeoJSON from NYC Open Data...")
    resp = requests.get(GEOJSON_URL, timeout=60)
    resp.raise_for_status()
    _log("INFO", f"Downloaded {len(resp.content):,} bytes")

    gdf = gpd.read_file(io.BytesIO(resp.content))
    _log("INFO", f"Loaded {len(gdf)} total NTA features (5 boroughs)")

    # Verify CRS — Socrata exports in WGS84
    assert gdf.crs is not None, "GeoJSON has no CRS"
    if gdf.crs.to_epsg() != 4326:
        _log("WARN", f"CRS is {gdf.crs}; reprojecting to EPSG:4326")
        gdf = gdf.to_crs("EPSG:4326")

    # Filter to Manhattan (BoroCode "1") and Brooklyn (BoroCode "3")
    # BoroCode is a string in Socrata exports — do NOT compare to int 1 or 3
    filtered = gdf[gdf["BoroCode"].isin(["1", "3"])].copy()
    if len(filtered) == 0:
        raise ValueError(
            "BoroCode filter returned 0 rows. "
            "Check that BoroCode column exists and contains string values '1' and '3'."
        )
    _log("INFO", f"Filtered to {len(filtered)} NTAs (Manhattan + Brooklyn)")

    # Write filtered GeoJSON
    out = Path(output_path)
    out.parent.mkdir(parents=True, exist_ok=True)
    filtered.to_file(str(out), driver="GeoJSON")
    _log("INFO", f"Boundary GeoJSON written: {out}")
    return filtered


if __name__ == "__main__":
    try:
        gdf = download_nta_boundaries(str(OUTPUT_PATH))
        manhattan = gdf[gdf["BoroCode"] == "1"]
        brooklyn = gdf[gdf["BoroCode"] == "3"]
        _log("INFO", f"Manhattan NTAs: {len(manhattan)}")
        _log("INFO", f"Brooklyn NTAs: {len(brooklyn)}")
        _log("INFO", f"Total NTAs committed: {len(gdf)}")
    except Exception as exc:
        _log("FAIL", str(exc))
        sys.exit(1)
```

Run this to see all NTACodes, then write nta_name_curation.json. For NTAs with clean single names, pass them through unchanged. For hyphenated merges, pick the most recognisable component or a widely-used community name.

**Curation rules:**
1. If the NTA name is already a single, universally-recognised name (e.g., "Williamsburg", "SoHo"), use it as-is.
2. If the NTA name is a hyphenated merge like "SoHo-TriBeCa-Civic Center-Little Italy", pick the most prominent area: "SoHo".
3. If the NTA covers areas without a dominant single name, use a short combined form: "Carroll Gardens-Cobble Hill".
4. Do NOT split NTA geometries — one NTACode, one display name.
5. All boroughs should be consistent: Brooklyn names should feel Brooklyn, Manhattan names should feel Manhattan.

**Known awkward names and recommended curations (apply these, read actual GeoJSON to get exact NTACodes):**

Manhattan curations:
- "Financial District-Battery Park City" → "Financial District / Battery Park City"
- "SoHo-TriBeCa-Civic Center-Little Italy" → "SoHo-TriBeCa"
- "Chinatown-Two Bridges" → "Chinatown"
- "Manhattanville-West Harlem" → "West Harlem"
- "Morningside Heights-Manhattan Valley" → "Morningside Heights"
- "Hudson Yards-Chelsea-Flatiron-Union Square" → "Chelsea-Flatiron"
- "Gramercy-Murray Hill" → "Gramercy-Murray Hill" (acceptable)
- "Midtown South-Flatiron-Union Square" → "Midtown South"
- "Midtown-Midtown South" → "Midtown"
- "Lincoln Square" → "Lincoln Square" (fine)
- "Upper West Side" names — accept as-is if not split
- "Upper East Side" names — accept as-is if not split
- "Harlem (South)", "Harlem (North)", "East Harlem (South)", "East Harlem (North)" → remove parenthetical direction if the NTA clearly covers the common named area; otherwise keep parenthetical for disambiguation

Brooklyn curations:
- "Carroll Gardens-Cobble Hill-Gowanus-Red Hook" → "Carroll Gardens-Red Hook"
- "Crown Heights North-Prospect Lefferts Gardens-Wingate" → "Crown Heights North"

- "Bay Ridge-Dyker Heights" → "Bay Ridge"
- "Coney Island-Sea Gate" → "Coney Island"
- "Sheepshead Bay-Gerritsen Beach-Manhattan Beach" → "Sheepshead Bay"
- "Bensonhurst East" / "Bensonhurst West" → keep directional for disambiguation
- "Sunset Park East" / "Sunset Park West" → keep directional
