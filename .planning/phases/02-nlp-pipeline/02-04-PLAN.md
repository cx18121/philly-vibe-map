        and archetype seed phrase embeddings, weighted by review count per topic.

    2. **Recency weight function (public, reused by temporal.py):**
       ```python
       HALF_LIFE_DAYS = int(os.environ.get("VIBE_HALF_LIFE_DAYS", "365"))
       MIN_WEIGHT = 1e-6

       def compute_recency_weight(review_date: str, reference_date: str, half_life_days: int = HALF_LIFE_DAYS) -> float:
           """Compute exponential decay weight in log-space.

           Args:
               review_date: ISO date string "YYYY-MM-DD"
               reference_date: ISO date string for "now"
               half_life_days: decay half-life in days

           Returns:
               float weight in [MIN_WEIGHT, 1.0]
           """
           from datetime import datetime
           d_review = datetime.strptime(review_date[:10], "%Y-%m-%d")
           d_ref = datetime.strptime(reference_date[:10], "%Y-%m-%d")
           delta_days = (d_ref - d_review).days
           if delta_days <= 0:
               return 1.0
           decay_lambda = np.log(2) / half_life_days
           log_weight = -decay_lambda * delta_days
           weight = np.exp(log_weight)
           return float(max(weight, MIN_WEIGHT))
       ```

    6. **Compute per-neighbourhood vibe scores:**
       For each neighbourhood:
       a. Get all review rowids in that neighbourhood
       b. Get each review's topic assignment and recency weight
       c. Aggregate: for each topic present in the neighbourhood, compute weighted count
       d. For each archetype, compute weighted sum of cosine similarities between topic centroids and archetype centroid, weighted by topic review count
       e. Normalize to [0, 1] range

       Use the most recent review date in the DB as `reference_date` for recency weighting.

       ```python
       from sklearn.metrics.pairwise import cosine_similarity

       reference_date = max(date for _, (date, _) in review_meta.items())

       neighbourhood_scores = {}
       for nid in sorted(set(nid for _, (_, nid) in review_meta.items())):
           # Get reviews for this neighbourhood
           nid_reviews = [(rid, review_meta[rid]) for rid in review_meta if review_meta[rid][1] == nid]

           # Weighted topic distribution
           topic_weights = defaultdict(float)
           for rid, (rdate, _) in nid_reviews:
               tid = topic_assignments.get(str(rid))
               if tid is not None and tid != -1 and tid in topic_centroids:
                   w = compute_recency_weight(rdate, reference_date)
                   topic_weights[tid] += w

           # Vibe score per archetype
           scores = {}
           total_weight = sum(topic_weights.values())
           if total_weight == 0:
               scores = {a: 0.0 for a in archetypes}
           else:
               for arch_name, arch_centroid in archetype_centroids.items():
                   weighted_sim = 0.0
                   for tid, tw in topic_weights.items():
                       sim = cosine_similarity(
                           topic_centroids[tid].reshape(1, -1),
                           arch_centroid.reshape(1, -1)
                       )[0, 0]
                       weighted_sim += (tw / total_weight) * sim
                   scores[arch_name] = float(weighted_sim)

           neighbourhood_scores[nid] = scores
       ```
