       with open(artifacts_dir / "vibe_scores.json") as f:

    2. Build sorted neighbourhood ID list and vibe matrix:
       ```python
       sorted_nids = sorted(vibe_scores.keys())
       archetype_order = ["artsy", "foodie", "nightlife", "family", "upscale", "cultural"]
       vibe_matrix = np.array(

       )
       ```

    3. Build FAISS index with L2 normalization:
       ```python
       import faiss
       faiss.normalize_L2(vibe_matrix)  # in-place normalization for cosine similarity
       index = faiss.IndexFlatIP(6)  # 6 vibe dimensions, inner product
       index.add(vibe_matrix)
       faiss.write_index(index, str(artifacts_dir / "faiss_index.bin"))
       ```

    4. Save ID map:
       ```python
       faiss_id_map = {i: nid for i, nid in enumerate(sorted_nids)}
       json.dump(faiss_id_map, open(artifacts_dir / "faiss_id_map.json", "w"), indent=2)
       ```

    **Sub-stage B: Representative Quotes (NLP-08)**

    1. Load embeddings, review_ids, topic_assignments, archetype centroids.

    2. Load review texts from DB:
       ```python
       conn = sqlite3.connect(db_path)
       review_data = {}  # {rowid: (text, neighbourhood_id)}
       for row in conn.execute("""
           SELECT r.rowid, r.text, b.neighbourhood_id
           FROM reviews r
           JOIN businesses b ON r.business_id = b.business_id
           WHERE b.neighbourhood_id IS NOT NULL
       """):
           review_data[row[0]] = (row[1], row[2])
       ```

    3. Embed archetype seed phrases (same as vibe_score.py):
       ```python
       model = SentenceTransformer("all-MiniLM-L6-v2")
       with open(Path(__file__).parent.parent / "archetypes.json") as f:
           archetypes = json.load(f)
       archetype_centroids = {}
       for name, phrases in archetypes.items():
           archetype_centroids[name] = model.encode(phrases).mean(axis=0)
       ```

    4. For each neighbourhood and archetype, find top 5 reviews by cosine similarity:
       ```python
       from sklearn.metrics.pairwise import cosine_similarity

       quotes = {}  # {nid: {archetype: [quote, ...]}}
       for nid in sorted_nids:
           quotes[nid] = {}
           # Get embedding indices for this neighbourhood
           nid_indices = [
               idx for idx, rid in enumerate(review_ids)
               if rid in review_data and review_data[rid][1] == nid
           ]
           if not nid_indices:
               for arch in archetypes:
                   quotes[nid][arch] = []
               continue

           nid_embeddings = embeddings[nid_indices]

           for arch_name, arch_centroid in archetype_centroids.items():
               sims = cosine_similarity(nid_embeddings, arch_centroid.reshape(1, -1)).flatten()
               top_k = min(5, len(sims))
               top_indices = np.argsort(sims)[-top_k:][::-1]  # descending

               arch_quotes = []
               for ti in top_indices:
                   orig_idx = nid_indices[ti]
                   rid = int(review_ids[orig_idx])
                   text = review_data[rid][0]
                   # Truncate to 300 chars
                   if len(text) > 300:
                       text = text[:297] + "..."
                   arch_quotes.append(text)

               # Ensure at least 3 quotes (pad with empty if needed -- shouldn't happen)
               quotes[nid][arch_name] = arch_quotes[:5]
       ```

    5. Save:
       `json.dump(quotes, open(artifacts_dir / "representative_quotes.json", "w"), indent=2)`

    **Sub-stage C: Enriched GeoJSON (NLP-09 partial)**

       import glob

       ```python
       for feature in geojson["features"]:
           nid = feature["properties"].get("NEIGHBORHOOD_NUMBER")
           if nid and nid in vibe_scores:
               feature["properties"]["vibe_scores"] = vibe_scores[nid]
               # Add dominant vibe
               scores = vibe_scores[nid]
               dominant = max(scores, key=scores.get)
               feature["properties"]["dominant_vibe"] = dominant
               feature["properties"]["dominant_vibe_score"] = scores[dominant]
       ```

    3. Save:
       `json.dump(geojson, open(artifacts_dir / "enriched_geojson.geojson", "w"), indent=2)`
